MKServerPinger Class Reference

Inherits from NSObject
Declared in MKServerPinger.h

Overview

MKServerPinger implements a pinger object that can ping and query Mumble servers for information typically shown in a server list.

Once a MKServerPinger object is created, it will continually ping the remote server until it is destroyed. Whenever the MKServerPinger receives a reply from the remote server, it will inform its delegate.
