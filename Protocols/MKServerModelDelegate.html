MKServerModelDelegate Protocol Reference

Declared in MKServerModel.h

Overview

MKServerModelDelegate is the delegate of MKServerModel. It is called to notify any registered delegates of events happening on the server, or of replies to previously sent messages.

serverModel:userMoved:toChannel:byUser:

Called when a user is moved to another channel. This is also called when a user changes the channel he resides in (in which case user is equivalent to mover).

- (void)serverModel:(MKServerModel *)model userMoved:(MKUser *)user toChannel:(MKChannel *)chan byUser:(MKUser *)mover

Parameters

model

The MKServerModel object in which this event originated.

user

The user that was moved.

chan

The channel to which user was moved to.

mover

The user that performed the user move. If the move was performed by the server, mover is nil.

Discussion

In case the server initiated the move, the mover is nil.
